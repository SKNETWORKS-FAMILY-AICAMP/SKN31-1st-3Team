## 📌 프로젝트 개요

### ⚡ EV Insight: 전기차 비용 분석 플랫폼 ⚡

> 전기차와 내연차의 유지비를 비교하고  
> **데이터 기반으로 전기차 전환의 경제성을 분석하는 서비스**

본 프로젝트는 전기차 등록 추이, 충전소 인프라 현황,  
유가 및 전기요금 데이터를 통합하여  
**전기차와 내연차의 실제 비용 차이**를 직관적으로 보여주는 대시보드입니다.

단순한 가격 비교를 넘어,  
사용자의 주행거리 및 차량 정보를 기반으로  
👉 **5년 기준 예상 절감액**을 계산하여  
전기차 전환이 실제로 이득인지 판단할 수 있도록 지원합니다.

---

## 🎯 주요 기능

- 📈 전기차 vs 내연차 등록 추이 시각화  
- ⛽ 유가 변동 및 전기요금 변화 분석  
- 🔌 지역별 충전소 인프라 현황 (히트맵)  
- 💰 사용자 맞춤 5년 유지비 및 절감액 계산  
- 📊 데이터 기반 비용 비교 대시보드 제공
